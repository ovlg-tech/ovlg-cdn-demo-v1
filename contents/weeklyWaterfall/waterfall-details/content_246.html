Do you travel frequently? If so, then you must be using public Wi-Fi a lot. Consider using a Virtual Private Network (VPN) since it would help you avoid identity-theft. A VPN makes an encrypted connection between its provider's network and your device. You can access the Internet but can keep your exchanges private when using Wi-Fi. Usually, you have to pay an amount for using a VPN. Few VPNs are more secure to use than others, so try to shop around. Read all the reviews and then opt for the best one.
